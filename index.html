Eclipse instructions 1.Create new workspace, preferably named: TXE 1.6.5 Forked 2.Make a new project named TXE 3.Drag all of the files that you have downloaded from GitHub and Dropbox into the project folder, from either finder or Windows explorer. 4.Download the Swing Layout Extentions 5.Download the bouncy castle jar for the encryption file from http://www.bouncycastle.org/latest_releases.html 6.Right click on project in the explorer and click on configure build path 7.Add the jar downloaded from http://www.bouncycastle.org/latest_releases.html as an EXTERNAL JAR 8.Debug and run it should work 9.Create your versions of TXE 10.Please share or report your versions of TXE to the community.
